# Лабораторная работа №5: Анализ последовательностей

## Описание
Реализация алгоритмов AprioriAll и PrefixSpan для поиска последовательных паттернов в транзакциях интернет-магазина.

## Задания
1. Реализация AprioriAll с нуля
2. Анализ реальных данных (Online Retail)
3. Сравнение алгоритмов SPM
4. Предсказание следующего события
5. Контрастные последовательности
